from_photo_to_word

图片转文字

本项目包含分别针对 QPython（手机）和 IDLE（电脑）的 Python 代码，可将图片中的文字转换为可编辑文本。

QPython（手机版）使用说明:

  • 在 QPython 中安装所需库：pip install pytesseract pillow
  • 需要下载 Tesseract OCR 语言数据包（中文+英文）
  • 确保给予 QPython 相机和存储权限

IDLE（电脑版）使用说明:

  • 安装所需库：pip install pytesseract pillow
  • 下载并安装 Tesseract OCR：https://github.com/tesseract-ocr/tesseract
  • 下载中文语言包（chi_sim）

通用注意事项

  • 两种版本都需要 Tesseract OCR 引擎支持
  • 中文识别需要下载中文语言包（chi_sim）
  • 识别准确率取决于图片质量和文字清晰度
  • 对于复杂排版或手写文字，识别率可能会降低
  • 可以根据需要修改代码，例如添加更多语言支持或调整识别参数
